## 🚀 Déploiement Express (Vercel - 5 min)

### 1. Créer un compte
→ [vercel.com](https://vercel.com) (gratuit)

### 2. Push sur GitHub
```bash
git init
git add .
git commit -m "Initial commit - DataFuse Studio"
git remote add origin VOTRE_REPO_URL
git push -u origin main
```

### 3. Déployer
- Connectez-vous sur Vercel
- Importez votre repo GitHub
- Cliquez sur "Deploy"
- **Site en ligne en 2 minutes ! 🎉**
